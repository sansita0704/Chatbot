# Calling Ollama Models

## TinyLlama

TinyLlama is a lightweight version of the Llama model.

### Install

Run:

```bash
ollama run tinyllama
```

- This downloads the model if it is not already installed.
- Ollama also starts a local web server that allows applications to communicate with the model using HTTP requests.

### Ollama JavaScript Client

> https://github.com/ollama/ollama-js

- Instead of sending HTTP requests manually, we can use the official Ollama JavaScript client.
- It acts as a wrapper around the local server and abstracts the underlying HTTP communication.

Install it in the server directory:

```bash
bun add ollama
```

### Update `client.ts`

Import:

```ts
import { Ollama } from "ollama";
```

Create an Ollama client:

```ts
const ollamaClient = new Ollama();
```

Update `summarizeReviews()`:

```tsx
async summarizeReviews(reviews: string) {
    const response = await ollamaClient.chat({
        model: "tinyllama",
        messages: [
            {
                role: "system",
                content: summarizePrompt,
            },
            {
                role: "user",
                content: reviews,
            },
        ],
    });

    return response.message.content;
}
```
