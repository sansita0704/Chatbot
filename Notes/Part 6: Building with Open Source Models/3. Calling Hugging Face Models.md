# Calling Hugging Face Models

## Create an Access Token

1. Go to your Hugging Face profile.
2. Open **Settings**.
3. Go to **Access Tokens**.
4. Click **Create new access token**.

- Choose the **Read** tab. This creates a token with read access to your resources.

Add the token to `.env` file:

```env
HF_TOKEN=<token>
```

## Install in the Server Directory

```bash
bun add @huggingface/inference
```

- `@huggingface/inference` is the service provided by Hugging Face for hosting models.
- We will be using its free tier.

## Update the LLM Client

Update `llm/client.ts`:

```ts
import { InferenceClient } from "@huggingface/inference";
```

Create an inference client using the access token:

```ts
const inferenceClient = new InferenceClient(process.env.HF_TOKEN);
```

Create a new method for summarization:

```tsx
async summarize(text: string) {
    const output = await inferenceClient.summarization({
        model: "facebook/bart-large-cnn",
        inputs: text,
        provider: "hf-inference",
    });

    return output.summary_text;
}
```

## Update the Review Service

```ts
const summary = await llmClient.summarize(joinedReviews);
```

### Note

- This model is good at summarizing news articles, but it is not well-suited for summarizing a collection of customer reviews.
- As a result, it cannot effectively extract the key positive and negative themes about a product.
