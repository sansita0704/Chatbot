# Choosing the Right Model for the Job

We have two options:

1. Use a model fine-tuned specifically for review summarization.
2. Use a general-purpose model.

## Use a General-Purpose Model

Update `client.ts`:

```tsx
const chatCompletion = await inferenceClient.chatCompletion({
    model: "meta-llama/Llama-3.1-8B-Instruct:novita",
    messages: [
        {
            role: "user",
            content: "What is the capital of France?",
        },
    ],
});
```

- Here, `messages` is an array of messages.
- The first message can be a **system** message that gives instructions to the model.
- We will provide our prompt using the system message.

Update `prompts/summarize-reviews.txt`:

```text
Summarize the following customer reviews into a short paragraph highlighting key themes, both positive and negative:
```

We no longer need the `{{reviews}}` placeholder because the reviews will be sent as the second message.

### Update `client.ts`

Import the prompt:

```ts
import summarizePrompt from "../prompts/summarize-reviews.txt";
```

Update the `messages` array:

```tsx
messages: [
    {
        role: "system",
        content: summarizePrompt,
    },
    {
        role: "user",
        content: reviews,
    },
];
```

## Final Function

```tsx
async summarizeReviews(reviews: string) {
    const chatCompletion = await inferenceClient.chatCompletion({
        model: "meta-llama/Llama-3.1-8B-Instruct:novita",
        messages: [
            {
                role: "system",
                content: summarizePrompt,
            },
            {
                role: "user",
                content: reviews,
            },
        ],
    });

    return chatCompletion.choices[0]?.message.content || "";
}
```
