# Using Hugging Face Models with Ollama

1. Go to your Hugging Face profile.
2. Open **Settings**.
3. Go to **Local Apps** from the left sidebar.
4. Enable **Ollama**.

- Ollama can only run Hugging Face models that support the **GGUF** format.
- Filter models by **GGUF** to see the ones compatible with Ollama.

After enabling Ollama, every compatible model page on Hugging Face will show **Ollama** in the **Use this model** dropdown.

From there, we can copy the command required to install that model.

Example:

```bash
ollama run hf.co/lmg-anon/vntl-llama3-8b-v2-gguf:Q5_K_M
```

After running the command, we can interact with that model locally.
