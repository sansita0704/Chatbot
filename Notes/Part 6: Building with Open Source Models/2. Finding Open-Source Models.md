# Finding Open Source Models

## Hugging Face

> [huggingface.co](https://huggingface.co)

- **Hugging Face** is the primary platform for hosting and discovering open-source AI models.
- It can be thought of as the **GitHub for machine learning**.
- Developers and researchers use it to publish:
    - Open-source models
    - Datasets

### Viewing Sample Code

There are **two approaches** with Javascript.

#### Option 1: Using `fetch`

Use JavaScript's `fetch` (or Axios) to directly call the model endpoint.

Workflow:

1. Send an HTTP POST request.
2. Include an **Authorization** header containing your Hugging Face token.
3. Receive the summarized text in the response.

#### Option 2: Using HuggingFace.js

Instead of manually sending HTTP requests, use the **HuggingFace.js** library.

Purpose:

- Abstracts the underlying HTTP communication.
- Provides a higher level of abstraction.

Instead of dealing with HTTP requests directly, you work with library methods.

Basic workflow:

1. Import the `InferenceClient` class.
2. Create an instance of the client.
3. Call a utility method such as `summarization()`.
4. Provide:
    - Model name
    - Input text
    - Provider

5. Receive the generated summary.

This approach is simpler than manually using `fetch`.
