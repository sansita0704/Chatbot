# Sending Data to the Server

- To send prompts to our backend, we need to make an HTTP request.
- There are two common options: `fetch()` or Axios

We will use **Axios**.

## Install Axios

Inside the client directory:

```bash
bun add axios
```

Import it in `ChatBot.tsx`:

```ts
import axios from "axios";
```

## Sending a POST Request

```ts
const onSubmit = async ({ prompt }: FormData) => {
    reset();

    const { data } = await axios.post("/api/chat", {
        prompt,
        conversationId: conversationId.current,
    });
};
```

### Why Reset First?

- We clear the form **before** making the network request.

Why?

- Calling the backend may take time.
- If we waited for the request to complete first, the textarea would continue showing the previous message, making the UI feel less responsive.
- Resetting immediately gives instant feedback to the user.

### Understanding `axios.post()`

```ts
axios.post(url, data);
```

Arguments:

1. URL: The API endpoint.
2. Request Body: The data to send.

### Conversation ID

- Every conversation should have a unique identifier.
- Generate it once when the component is created.

```tsx
const conversationId = useRef(crypto.randomUUID());
```

### `crypto.randomUUID()`

- Modern browsers provide a built-in `crypto` object.
- It includes `crypto.randomUUID()` which generates a UUID.
- This ID uniquely identifies the current conversation.

### Why Use `useRef`?

- We could store the conversation ID inside state.
- However, state is unnecessary here.

The conversation ID:

- is created once
- never changes
- should not trigger re-renders

`useRef` is designed for exactly this purpose.

- With `useRef`, we can store values that should not cause a re-render.

## Final `ChatBot.tsx`

```tsx
// Import 3rd-party code first:
import { useRef, type KeyboardEvent } from "react";
import axios from "axios";
import { FaArrowUp } from "react-icons/fa";
import { useForm } from "react-hook-form";

// Then import our own code:
import { Button } from "./ui/button";

type FormData = {
    prompt: string;
};

const ChatBot = () => {
    const conversationId = useRef(crypto.randomUUID());

    const { register, handleSubmit, reset, formState } = useForm<FormData>();

    // Destructure the data here only:
    const onSubmit = async ({ prompt }: FormData) => {
        // Clear the form immediately
        reset();

        // Send the message to the backend
        const { data } = await axios.post("/api/chat", {
            prompt,
            conversationId: conversationId.current,
        });

        console.log(data);
    };

    const onKeyDown = (e: KeyboardEvent<HTMLFormElement>) => {
        if (e.key === "Enter" && !e.shiftKey) {
            e.preventDefault();

            handleSubmit(onSubmit)();
        }
    };

    return (
        <form
            onSubmit={handleSubmit(onSubmit)}
            onKeyDown={onKeyDown}
            className="flex flex-col gap-2 items-end border-2 p-4 m-2 rounded-2xl"
        >
            ...
        </form>
    );
};

export default ChatBot;
```
