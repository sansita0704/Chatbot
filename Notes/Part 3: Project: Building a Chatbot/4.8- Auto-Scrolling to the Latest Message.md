# Auto-Scroll to the Latest Message

- As the conversation grows, new messages appear below the visible portion of the screen.
- Without auto-scrolling, users must manually scroll down after every new message.
- A better experience is to automatically scroll to the latest message.

## Create a Reference

We'll scroll to the form whenever a new message is added.

Create a reference:

```tsx
const formRef = useRef<HTMLFormElement | null>(null);
```

### Why `HTMLFormElement | null`?

- During the initial render, the form does not exist yet.
- Therefore, the reference is initialized as `null`.
- Once React mounts the form, `formRef.current` points to the actual DOM element.

### Attach the Reference

```tsx
<form
    ref={formRef}
    onSubmit={handleSubmit(onSubmit)}
    onKeyDown={onKeyDown}
    className="flex flex-col gap-2 items-end border-2 p-4 m-2 rounded-2xl"
>
```

- Now `formRef.current` references the form element.

## Scroll Using `useEffect`

Whenever the `messages` array changes:

```tsx
useEffect(() => {
    formRef.current?.scrollIntoView({
        behavior: "smooth",
    });
}, [messages]);
```

- With `formRef.current?.scrollIntoView()`, we can scroll the page to bring the form into view.
- To make the scrolling animation smooth, we pass the options object:

```tsx
{
    behavior: "smooth",
}
```
