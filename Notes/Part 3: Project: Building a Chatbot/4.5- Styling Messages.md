# Styling Messages

Create a dedicated type for each message:

```tsx
type Message = {
    content: string;
    role: "user" | "bot";
};
```

The `role` property lets us distinguish between:

- User messages
- AI messages

## Update the State

```tsx
const [messages, setMessages] = useState<Message[]>([]);
```

### Add a User Message

```tsx
setMessages((prev) => [
    ...prev,
    {
        content: prompt,
        role: "user",
    },
]);
```

### Add the AI Response

```tsx
setMessages((prev) => [
    ...prev,
    {
        content: data.message,
        role: "bot",
    },
]);
```

### Render the Message Content

```tsx
<p key={index}>{message.content}</p>
```

## Dynamic Styling

Now that every message has a `role`, we can style user and bot messages differently.

```tsx
<p
    key={index}
    className={`px-3 py-1 rounded-xl ${
        message.role === "user" ?
            "bg-blue-600 text-white self-end"
        :   "bg-gray-100 text-black self-start"
    }`}
>
    {message.content}
</p>
```
