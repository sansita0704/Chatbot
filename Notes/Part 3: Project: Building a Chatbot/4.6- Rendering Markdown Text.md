# Rendering Markdown

- AI responses often contain Markdown.
- To properly render Markdown, use the **react-markdown** library.

## Install

Inside the client directory:

```bash
bun add react-markdown
```

### Import

```tsx
import ReactMarkdown from "react-markdown";
```

### Render Markdown

Wrap the message content inside `ReactMarkdown`.

```tsx
<ReactMarkdown>{message.content}</ReactMarkdown>
```
