# Building the Chat UI

## Step 1: Create `ChatBot.tsx`

Create:

```text
components/ChatBot.tsx
```

```tsx
import { Button } from "./ui/button";

const ChatBot = () => {
    return (
        <div>
            <textarea />
            <Button>Send</Button>
        </div>
    );
};

export default ChatBot;
```

## Step 2: Update `App.tsx`

```tsx
import ChatBot from "./components/ChatBot";

function App() {
    return (
        <div>
            <ChatBot />
        </div>
    );
}

export default App;
```

## Styling the Chatbot

Update the component:

```tsx
import { Button } from "./ui/button";
import { FaArrowUp } from "react-icons/fa";

const ChatBot = () => {
    return (
        <div className="flex flex-col gap-2 items-end border-2 p-4 m-2 rounded-2xl">
            <textarea
                className="w-full border-0 outline-0 resize-none text-sm"
                placeholder="Ask anything"
                maxLength={1000}
            />
            <Button className="cursor-pointer rounded-full w-9 h-9">
                <FaArrowUp />
            </Button>
        </div>
    );
};

export default ChatBot;
```

### Why Set Both Width and Height?

Suppose we write:

```tsx
<Button className="cursor-pointer rounded-full">
    <FaArrowUp />
</Button>
```

- The button **does not become a perfect circle**.
- This is because buttons are rectangular by default.

To create a circle, the element must also have equal width and height:

```tsx
<Button className="cursor-pointer rounded-full w-9 h-9">
    <FaArrowUp />
</Button>
```

Result:

- A perfectly circular button.
