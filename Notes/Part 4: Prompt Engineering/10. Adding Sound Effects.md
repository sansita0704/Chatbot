# Adding Sound Effects

## Create a sounds folder

Create the following directory:

```
client/
└── src/
    └── assets/
        └── sounds/
            ├── pop.mp3
            └── notification.mp3
```

### Why is the sound logic inside `ChatBot.tsx`?

The audio logic belongs in `ChatBot.tsx` because this component is responsible for the entire chat experience.

It:

- Displays the conversation
- Handles message submission
- Shows the typing indicator
- Receives the bot response

Since it coordinates all of these events, it is the ideal place to trigger sound effects.

### Import the sound files

```ts
import popSound from "@/assets/sounds/pop.mp3";
import notificationSound from "@/assets/sounds/notification.mp3";
```

> `@` points to the project's root source directory.

### Create audio objects

Create an `Audio` object for each sound.

```ts
const popAudio = new Audio(popSound);
popAudio.volume = 0.2;

const notificationAudio = new Audio(notificationSound);
notificationAudio.volume = 0.2;
```

### Play sounds in the submit handler

Update the `onSubmit` handler:

```tsx
try {
    setError("");
    setMessages((prev) => [
        ...prev,
        {
            content: prompt,
            role: "user",
        },
    ]);

    setIsBotTyping(true);
    popAudio.play();

    ...

    setMessages((prev) => [
        ...prev,
        {
            content: data.message,
            role: "bot",
        },
    ]);
    notificationAudio.play();
}
```
