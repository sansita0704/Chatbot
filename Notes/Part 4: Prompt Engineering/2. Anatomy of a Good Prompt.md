# Anatomy of a Good Prompt

Most prompts that produce **consistent, high-quality results** follow a simple structure:

1. **Instruction** - Tell the model what to do.
2. **Context** - Provide background information.
3. **Output Format** - Specify how the response should be returned.

## 1. Instruction

The **instruction** tells the AI exactly what task to perform.

### Basic Example

```text
Summarize the following product reviews.
```

This works, but it is somewhat vague.

### Improved Example

```text
Summarize the following reviews in three short bullet points using simple language.
```

This improved prompt clearly specifies:

- The task: summarize
- The number of points: three
- The format: bullet points
- The writing style: simple language

## 2. Context

**Context** provides background information that helps the model better understand the task.

Context can include:

- A role for the AI
- Text to analyze
- Background information
- Additional details about the task

### Example

```text
You're a senior software engineer.
Read the code snippet below and explain it in plain English.
```

Here, the context tells the model:

- What role to assume (senior software engineer)
- What content to work with (code snippet)
- What style to use (plain English)

### Why Context Matters

The clearer and more detailed the context, the better the model understands the task and the higher the quality of the output.

## 3. Output Format

The **output format** specifies how the response should be structured.

Possible formats include:

- Paragraph
- Bullet list
- Table
- JSON
- Other structured formats

### Example

```text
Label this message as spam or not spam.
Return the result as JSON with a single key called "label".
```

Now the model knows:

- What task to perform
- Exactly how the answer should be formatted

## Combining All Three Components

```text
You are a helpful support agent.
Summarize the following customer reviews in two to three bullet points.
Focus on pain points related to the login experience.
```

### Breakdown

**Context**

```text
You are a helpful support agent.
```

Defines the AI's role.

**Instruction**

```text
Summarize the following customer reviews.
```

Specifies the task.

**Additional Guidance**

```text
Focus on pain points related to the login experience.
```

Narrow the focus to relevant information.

**Output Format**

```text
Two to three bullet points
```

Specifies how the response should be presented.

**Return Type**

- Plain text
