# Handling Errors

To gracefully handle API failures while fetching reviews, create an error state.

```tsx
const [error, setError] = useState("");
```

## Update `fetchReviews()`

Wrap the API call inside a `try...catch...finally` block.

```tsx
const fetchReviews = async () => {
    try {
        setIsLoading(true);

        const { data } = await axios.get<GetReviewsResponse>(
            `/api/products/${productId}/reviews`,
        );

        setReviewData(data);
    } catch (error) {
        console.error(error);
        setError("Could not fetch the reviews. Try again!");
    } finally {
        setIsLoading(false);
    }
};
```

### Flow

1. Wrap the entire API call inside a `try` block.
2. If an error occurs, catch it and update the error state.
3. The `finally` block always executes, whether the request succeeds or fails, ensuring `isLoading` is reset to `false`.

### Render the Error

Display the error message if one exists.

```tsx
if (error) return <p className="text-red-500">{error}</p>;
```
