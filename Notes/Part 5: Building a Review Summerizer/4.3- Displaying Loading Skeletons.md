# Displaying Loading Skeletons

We'll use the `react-loading-skeleton` library to display a loading skeleton.

### 1. Create a Loading State

Track whether the reviews are currently being fetched.

```tsx
const [isLoading, setIsLoading] = useState(false);
```

### 2. Update `fetchReviews()`

Set the loading state before and after the API call.

```tsx
const fetchReviews = async () => {
    setIsLoading(true);

    const { data } = await axios.get<GetReviewsResponse>(
        `/api/products/${productId}/reviews`,
    );

    setReviewData(data);
    setIsLoading(false);
};
```

Initially, we could simply render:

```tsx
if (isLoading) return <p>Loading...</p>;
```

### 3. Install the Library

```bash
bun add react-loading-skeleton
```

### 4. Import the CSS

In `main.tsx`:

```tsx
import "react-loading-skeleton/dist/skeleton.css";
```

### 5. Import the Component

Inside `ReviewList.tsx`:

```tsx
import Skeleton from "react-loading-skeleton";
```

### 6. Display Skeletons

Replace the loading text with skeleton placeholders.

```tsx
if (isLoading)
    return (
        <div className="flex flex-col gap-5">
            {[1, 2, 3].map((i) => (
                <div key={i}>
                    <Skeleton width={150} />
                    <Skeleton width={100} />
                    <Skeleton count={2} />
                </div>
            ))}
        </div>
    );
```

Three skeleton blocks are rendered, each representing a review.

Each review consists of:

1. Author name
2. Rating
3. Review content
