# Setting Up MySQL

We need a database to store our product reviews and AI-generated summaries.

## MySQL

- **MySQL** is an **open-source relational database management system (RDBMS)**.

### Installation

Go to the MySQL Downloads page and install MySQL Community Server.

- Download from: [https://www.mysql.com/downloads/](https://www.mysql.com/downloads/)
- Click **MySQL Community Downloads**
- Select **MySQL Community Server**
- Choose the appropriate version for your operating system and install it.
