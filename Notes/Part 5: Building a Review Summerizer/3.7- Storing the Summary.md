# Storing the Summary

- Currently, we generate a new summary every time the API is called.
- This approach is slow and expensive.
- Every API call sends a request to the AI model, consuming tokens and increasing cost.

Instead, we will **store the generated summary in the database**.

This way:

- The summary is generated only once.
- Future requests return the cached summary from the database.

To achieve this, we need to add a new repository method.

## Update Repository

Create a new function:

```ts
storeReviewSummary(productId: number, summary: string) {}
```

### Using `upsert()`

- We will use Prisma's `upsert()` method.

**Upsert = Insert + Update**

- If the record does **not** exist, Prisma inserts it.
- If the record **already exists**, Prisma updates it.

`upsert()` accepts an object with three properties.

### 1. `where`

- Used to identify the record.
- It specifies which record should be inserted or updated.

### 2. `create`

- Contains the data to insert if no matching record exists.

### 3. `update`

- Contains the data to update if the record already exists.

```ts
storeReviewSummary(productId: number, summary: string) {
    const now = new Date();
    const expiresAt = dayjs().add(7, "days").toDate();

    const data = {
        content: summary,
        expiresAt,
        generatedAt: now,
        productId,
    };

    return prisma.summary.upsert({
        where: { productId },
        create: data,
        update: data,
    });
},
```

- `upsert()` is an asynchronous method.
- Therefore, we return the promise here and `await` it in the service.

### Using `dayjs`

- We need to calculate the `expiresAt` value.

Install `dayjs` inside the **server** directory:

```bash
bun add dayjs
```

Import it:

```ts
import dayjs from "dayjs";
```

Calculate the expiration date:

```ts
const expiresAt = dayjs().add(7, "days").toDate();
```

### `dayjs()`

- Returns the current date and time.
- However, the returned object is not a native JavaScript `Date` object.

### `.add(7, "days")`

- Adds 7 days to the current date.

### `.toDate()`

- Converts the Day.js object into a native JavaScript `Date` object.

## Update the Service

After receiving the summary from the AI model, store it in the database before returning it.

```ts
const summary = response.text;

await reviewRepository.storeReviewSummary(productId, summary);

return summary;
```

Or, using object destructuring:

```ts
const { text: summary } = await llmClient.generateText({
    messages: [
        {
            role: "user",
            content: prompt,
        },
    ],
    maxTokens: 500,
});

await reviewRepository.storeReviewSummary(productId, summary);

return summary;
```

Here, we directly destructure the `text` property and rename it to `summary`.
