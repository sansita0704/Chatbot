# Handling Errors

Create a state variable to track summary generation errors.

```tsx
const [summaryError, setSummaryError] = useState("");
```

### Reset Previous Errors

Before making a new request, clear any previous error.

```tsx
setSummaryError("");
```

This ensures that if the user retries, the previous error message is removed.

### Update `handleSummarize()`

```tsx
const handleSummarize = async () => {
    try {
        setIsSummaryLoading(true);
        setSummaryError("");

        const { data } = await axios.post<SummarizeResponse>(
            `/api/products/${productId}/reviews/summarize`,
        );

        setSummary(data.summary || "");
    } catch (error) {
        console.error(error);
        setSummaryError("Could not summarize the reviews. Try again!");
    } finally {
        setIsSummaryLoading(false);
    }
};
```

### Flow

1. Wrap the request inside a `try...catch...finally` block.
2. Reset any previous error before sending the request.
3. If an error occurs, update the `summaryError` state.
4. Use the `finally` block to reset `isSummaryLoading`, regardless of whether the request succeeds or fails.

## Render the Error

Display the error message below the button.

```tsx
{
    summaryError && <p className="mt-2 text-red-500">{summaryError}</p>;
}
```
