# Extracting the Prompt

- Instead of hardcoding the prompt inside the service, we can move it to a separate file.

Create a new file:

```text
prompts/summarize-reviews.txt
```

Contents:

```text
Summarize the following customer reviews into a short paragraph highlighting key themes, both positive and negative:

{{reviews}}
```

`{{reviews}}` acts as a placeholder that will later be replaced with the actual reviews.

## Update the Review Service

Import the prompt template:

```ts
import template from "../prompts/summarize-reviews.txt";
```

Replace the placeholder with the reviews:

```ts
const prompt = template.replace("{{reviews}}", joinedReviews);
```
