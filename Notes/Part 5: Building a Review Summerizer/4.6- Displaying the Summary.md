# Displaying the Summary

Add a new section above the reviews inside `ReviewList`.

```tsx
<div className="mb-5">
    {reviewData?.summary ?
        <p>{reviewData.summary}</p>
    :   <Button>
            <HiSparkles />
            Summarize
        </Button>
    }
</div>
```

- If a summary already exists, display it.
- Otherwise, display the **Summarize** button.

### No Reviews

If a product has no reviews, there is no reason to show the **Summarize** button.

Instead, render:

```tsx
if (!reviewData?.reviews.length)
    return <p>No reviews found for this product.</p>;
```
