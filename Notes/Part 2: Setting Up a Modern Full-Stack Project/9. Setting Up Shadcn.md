# shadcn/ui

- **shadcn/ui** is a modern UI component library built on top of:
    - Radix UI
    - Tailwind CSS

- It provides a collection of beautifully designed, accessible, and highly customizable UI components.

## Install shadcn/ui

Documentation:

```text
https://ui.shadcn.com/docs/installation/vite#existing-project
```

Follow all steps provided in the documentation.

### Important Notes

Run all installation commands inside the **client directory**.

For example:

```bash
bun add -D @types/node
```

### Bun Equivalents

- npm → bun
- npx → bunx

## Adding a Button Component

Inside the client directory:

```bash
bunx --bun shadcn@latest add button
```

## Using the Button

Example:

```tsx
return (
    <div className="p-4">
        <p className="font-bold p-4 text-2xl">{message}</p>
        <Button className="ml-4">Click me!</Button>
    </div>
);
```
