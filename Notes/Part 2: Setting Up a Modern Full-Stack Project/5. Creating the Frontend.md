# Creating the Frontend

For the frontend application, we will use **Vite**.

## What is Vite?

Vite is a modern frontend build tool that provides:

- Fast development server
- Hot Module Replacement (HMR)
- Optimized production builds
- Excellent support for React, Vue, and other frameworks

## Creating a New Vite Project

Official documentation:

```text
https://vite.dev/guide/
```

Inside the client directory, run:

```bash
bun create vite .
```

### Why the Dot (`.`)?

Without the dot:

```bash
bun create vite
```

Vite creates a new subdirectory and places the project inside it.

With:

```bash
bun create vite .
```

Vite creates the project directly inside the current directory.

#

At this point, we have three `package.json` files:

1. Inside root directory
2. Inside server directory
3. Inside client directory

### Install Dependencies

Inside the client directory:

```bash
bun i
```

### Run the Frontend

```bash
bun run dev
```

The `dev` script is defined inside the client's `package.json`.
